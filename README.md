# backup-switch
script for backup. You need to have a csv file with ip, type, user, password and directory of backup to your devices.

pip install -r requirements.txt
python3 backup_device.py listado.csv

Is support:
    - cisco_ios 
    - cisco_s300
    - cisco_wlc
    - huawei
    - smartzone
    - vyos
    - fortinet
